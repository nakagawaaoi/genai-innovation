# 生成AIイノベーション合同会社 コーポレートサイト

新潟を拠点に、生成AIの社会実装とAX（AIトランスフォーメーション）を推進する「生成AIイノベーション合同会社」の公式コーポレートサイトのソースコードです。

## 🌐 公開URL (GitHub Pages)
※ここに公開後のURLを記載します（例: `https://<あなたのGitHubユーザー名>.github.io/genai-innovation-web/`）

## 🚀 特長
*   **レスポンシブデザイン**: スマートフォン、タブレット、PCのすべてのデバイスに最適化。
*   **モダンなデザインシステム**: 新潟の自然（山・川・緑）と先進テクノロジー（AI）の融合をテーマにした、エメラルドグリーン基調のプレミアムなダークテーマ。
*   **パフォーマンス最適化**: 軽量なVanilla HTML/CSS/JSで構成され、高速な読み込みを実現。
*   **SEO対応**: タイトルタグ、メタディスクリプション、セマンティックHTMLを最適化済み。

## 📂 フォルダ構成
*   `index.html` : サイトの構造とテキストコンテンツ
*   `style.css` : 配色、レイアウト、アニメーション等のスタイルシート
*   `main.js` : スクロールアニメーション、お問い合わせフォームの挙動
*   `assets/logo.png` : 新潟の山とデータ波を融合した公式ロゴマーク（透過PNG）
